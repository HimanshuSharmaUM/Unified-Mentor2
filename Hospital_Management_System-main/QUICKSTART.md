# Operation Scheduler - Quick Start Guide

This guide helps you get the Operation Scheduler running quickly.

## Prerequisites
- Python 3.7 or higher
- pip (Python package manager)

## Quick Start (5 steps)

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Run the Application
```bash
python app.py
```

### 3. Access the Application
Open your browser and visit: `http://127.0.0.1:5000`

### 4. Start Using
- Dashboard shows all module counts
- Click any tile to manage that section
- Add, edit, or delete records as needed

## For Offline Demo
Simply open `static/index.html` in your browser to see the interface without running the server.

## Configuration
Edit `config.json` to change:
- Database name (default: database.db)
- Port (default: 5000)
- Host (default: 127.0.0.1)
